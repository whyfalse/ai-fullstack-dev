# setting.local.json 规则说明

本项目存在两类 `setting.local.json`，分别位于**全栈项目管理目录**和**前端/后端代码项目**下，记录彼此的路径映射。

## 1. 全栈项目管理目录下的 `setting.local.json`

记录前端、后端代码项目的路径。

```json
{
    "frontend": {
        "projectPath": "D:/frontend-proj"
    },
    "backend": {
        "projectPath": "D:/backend-proj"
    }
}
```

## 2. 前端 / 后端代码项目下的 `setting.local.json`

记录全栈项目管理目录的根路径，供前端/后端 Agent 读取需求、契约、任务看板等文件。

```json
{
    "devManagerPath": "D:/ai-fullstack-dev"
}
```
