# `User`

用户基础信息

## 属性

| 名称           | 类型                | 描述       |
| ------------ | ----------------- | -------- |
| `id`         | integer(int64)    | 用户的唯一标识符 |
| `username`   | string            | 登录用户名    |
| `email`      | string            | 用户电子邮箱   |
| `nickname`   | string            | 对外展示的昵称  |
| `age`        | integer(int32)    | 用户年龄     |
| `createTime` | string(date-time) | 记录创建时间   |

**必需字段:** `username`
