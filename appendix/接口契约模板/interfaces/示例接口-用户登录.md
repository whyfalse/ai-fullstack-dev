# 用户登录

**接口信息**

- **标签:** `示例接口`
- **请求方法:** `POST`
- **接口路径:** `/api/example/login`
- **操作ID:** `login`

**描述:** 用户登录，成功后返回访问令牌

## 请求体

**Content Type:**

- `application/json`
  - **Schema:** [LoginRequest](../params/LoginRequest.md)

## 响应

### 200 - OK

**Content Type:**
- `*/*`
  - **Schema:** [ApiResponseString](../params/ApiResponseString.md)
