# 分页获取用户列表

**接口信息**

- **标签:** `示例接口`
- **请求方法:** `GET`
- **接口路径:** `/api/example/users`
- **操作ID:** `listUsers`

**描述:** 分页获取用户列表

## 请求参数

| 位置 | 名称 | 类型 | 必需 | 描述 |
|------|------|------|------|------|
| query | `page` | integer(int32) | 是 | 当前页码，从1开始 |
| query | `pageSize` | integer(int32) | 是 | 每页条数 |
| query | `keyword` | string | 否 | 用户名关键字，模糊匹配 |

## 响应

### 200 - OK

**Content Type:**
- `*/*`
  - **Schema:** [ApiResponsePageResponseUser](../params/ApiResponsePageResponseUser.md)
