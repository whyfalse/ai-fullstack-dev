## 文件目录结构

/
├── README.md                    # 项目简介，包含当前版本、项目背景等信息
├── setting.local.json                 # 个人设置，每个开发者单独进行配置
├── requirements/                # 需求管理目录
│   ├── docs/                    # 通用需求文档目录，存放PRD、原型图、质量管理等文档
│   ├── module/                  # 模块级需求目录
│   │   ├── catalog.md           # 需求模块目录
│   │   ├── <模块名称>/          # 单个模块需求文件夹
│   │   │   ├── README.md        # 需求模块文档
│   │   │   └── change-log/      # 变更日志
│   │   │       ├── 20260518-0.md # 变更记录（日期+序号格式）
│   │   │       └── ...
│   │   └── ...
│   └── ...
├── architecture/                # 架构设计目录
│   ├── README.md                # 架构设计说明，包含架构设计思维、前后端技术选型等内容
│   ├── module/                  # 分析需求后拆分出的程序模块
│   │   ├── catalog.md          # 程序功能模块目录
│   │   ├── <模块名称>/          # 单个程序模块
│   │   │   ├── frontend.md      # 前端功能文档
│   │   │   ├── backend.md       # 后端功能文档
│   │   │   ├── test.md          # 功能测试用例
│   │   │   └── change-log/      # 变更日志
│   │   │       ├── 20260518-0.md # 变更记录（日期+序号格式）
│   │   │       └── ...
│   │   └── ...
│   ├── contracts/               # 架构契约
│   │   ├── apis/                # API接口契约目录
│   │   │   ├── catalog.md       # API接口目录索引
│   │   │   ├── interfaces/      # 接口定义目录
│   │   │   │   ├── <接口名称>.md # 单个接口契约文档（含请求/响应定义）
│   │   │   │   └── ...
│   │   │   └── params/          # 出入参定义
│   │   │       ├── <参数类名>.md # 参数详情
│   │   │       └── ...
│   │   └── database/            # 数据库设计
│   │       ├── catalog.md       # 数据库设计目录索引
│   │       ├── <表名>.json      # 表/集合 schema 中立描述（JSON，按目标数据库映射为 DDL 或 schema 验证）
│   │       └── ...
│   └── ...
├── dev-tasks/                   # 开发任务拆分目录
│   ├── frontend.md              # 前端开发任务清单
│   └── backend.md               # 后端开发任务清单
└── appendix/                    # 附录目录（补充文档、参考资料等）
